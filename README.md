Az iskolai feladatokhoz használt dll.

Használatával nem írjuk meg minden órán a fájlbeolvasást és/vagy írást.

Üzleti logika független metódusokat használunk:
Olvasás: - List<List<string>> típusú függvény, paraméterei: fájlneve (string), adattagok száma (int), splitter (char), van-e fejléc (bool). Visszatér egy List<List<string>> típusú  "adatok" objektummal. 
Írás: void metódus ami vár egy fájlnevet, és egy List<string>-et amiben a kiirandó adatok találhatóak.
